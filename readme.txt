== Description ==

Sitemap Generator WP is the easy way to generate an XML or HTML or CSV sitemap to your wordpress blog.

For support check out <a href="http://daext.com">daext.com</a>

Plugin Features:

* Generate an XML or HTML or CSV sitemap and set the priority for each specific category ( posts, pages, categories, tags )

* Sitemap Generator WP is fully compliant with the XML sitemap standard and is able to generate XML sitemap with mandatory tags ( URLSET - URL - LOC ) and optional tags ( LASTMOD - PRIORITY )

* The XML sitemap generated by Sitemap Generator WP is perfect for Google.

== Installation ==

1. Upload the Sitemap Generator WP plugin to your blog and activate it. 
2. Go to the Sitemap menu, select Generator to generate the sitemaps or Options to set you preferences.
